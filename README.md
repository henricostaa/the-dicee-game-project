# Dicee Game

Projeto simples em HTML, CSS e JavaScript que simula o lançamento de dois dados
e anuncia o vencedor com base no maior número.

## Pré-requisitos

- Navegador moderno (Chrome, Firefox, Edge, Safari, etc.)

## Como executar

1. Abra o arquivo `dicee.html` no navegador.
2. Atualize a página para gerar novos valores nos dados.

## Estrutura do projeto

- `dicee.html`: marcação da página.
- `styles.css`: estilos visuais.
- `index.js`: lógica do sorteio e atualização da interface.
- `images/`: imagens dos dados.

## Como funciona

Ao carregar a página, o JavaScript sorteia dois números de 1 a 6, atualiza as
imagens dos dados e exibe o vencedor (ou empate) no título principal.
